Overview

This was built as part of a deliverable for the course Developing Data Products as part of the Coursera Data Science Specialization.

The app developed for the first part of the assignment demo is available at: https://josh813.shinyapps.io/finance/

Source code for ui.R and server.R files are available on the GitHub repo: https://github.com/Josh813/finance

How it works

Someone famous once said that having money with no sense of guidance is the same as being poor. I tend to agree. Mirroring the myraid of money tracking softwares available out there, this application is designed as a basis which you can take and further developed for your own customization.

As easy as ...

  1. To fill in your expenditures and savings into a csv file
  2. Load this file into your R console along with these ui.R and server.R files
  3. Start customizing your analytical dashboard
